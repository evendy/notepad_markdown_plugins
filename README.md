#   notepad++ 添加markdown插件

###  使用markdown插件，让编写的关键字代码可以高亮，知道代码是否正确。



1. 下载master.zip解压，并将theme-default\userDefinedLang-markdown.default.classic.xml存放到notepad++安装目录
1. 打开notepad++，语言-自定义语言格式-导入，导入刚刚的xml文件，语言支持添加成功。
1. 下载NppMarkdown.dll存放到（notepad++安装目录）\plugins
1. 打开notepad++，观察插件，多了一个下载NppMarkdown插件，插件添加成功
